# PR Command Center

A mission-critical PR intelligence platform for enterprise brands and crisis management teams.

## Tech Stack

- **Next.js 14** - React framework with App Router
- **TypeScript** - Type safety
- **Tailwind CSS** - Utility-first styling with custom dark theme
- **Framer Motion** - Purposeful animations
- **Recharts** - Data visualization
- **Zustand** - State management

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
npm start
```

## Project Structure

```
src/
├── app/              # Next.js App Router pages
├── components/       # React components
│   ├── ui/          # Base UI components
│   ├── layout/      # Layout components
│   ├── dashboard/   # Dashboard components
│   ├── risk/        # Risk monitoring components
│   ├── influencer/  # Influencer intelligence components
│   ├── ai/          # Explainable AI components
│   └── mentions/    # Mentions explorer components
├── lib/             # Utilities and mock data
├── store/           # Zustand state management
├── styles/          # Global styles
└── types/           # TypeScript type definitions
```

## Design Principles

- **Dark-first theme** with neon accents for meaning
- **Glassmorphism** cards with backdrop blur
- **Purposeful animations** that communicate state
- **High information density** without clutter
- **Command center aesthetic** inspired by Palantir, Dataminr, Bloomberg Terminal
